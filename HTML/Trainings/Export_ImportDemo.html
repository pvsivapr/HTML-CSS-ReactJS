Export Import Tutorial :

Helps to include functions, classes, variables from other ".js" files to current ".js" files

Best way is Using HTML script - module:

  • In html "script" tag add "type" as "module" instead of "application/javascrip" or else it throws an error saying "Cannot use import statement outside a module."
  • Also to use the "module" type you need a server, cannot be able to run directly on the browser without running a server
Alternate way is by Using Java Script:
  • Here we don't use "export" or "import" keywords instead we get the entire ".js" file code and include it into the current "HTML" file as "script" by using the following lines
    1. var someObject = document.createElement('script');
    2. someObject.src = './xxxxxx/xxxxxxxxxx/xxxxxx.js';
    3. document.head.appendChild(someObject);
  • Then in the current ".js" file you can directly use the classes by creating objects, and also can use the functions and variables
  • The possible issue is with the name duplicates, this should be handled with care
Alternate way is by Using HTML Script tag:
  • Here we don't use "export" or "import" keywords instead we get the entire ".js" file code and include it into the current "HTML" file by using "script" tag by following procedure
    1. create a script tag for the ".js" file you want to include before our main ".js" file
  • Then in the current ".js" file you can directly use the classes by creating objects, and also can use the functions and variables
  • The possible issue is with the name duplicates, this should be handled with care
  • The both alternate methods have the same things in diferent ways
